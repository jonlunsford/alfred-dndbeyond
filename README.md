## Alfred + D&D Beyond

Quickly open search queries in your default browser from Alfred.

#### Usage
```
-----------------------
> dnd ...             |
-----------------------
| Search everything...|
-----------------------
| Filter by...        |
-----------------------
  -- All
  -- Backgrounds, Classes, Feats, Races
  -- Spells
  -- Monsters
  -- Items
  -- Vehicles
```
